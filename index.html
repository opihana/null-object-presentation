Sass

  • Sass is a scripting language ("Syntactically Awesome Stylesheets") that gets interpreted into CSS
  • Original syntax (.sass) is similar to Haml - indentation
  • Newer syntax (.scss) is similar to CSS - { }
  • Written in Ruby
